Introduction

This page includes supplementary material for the substitutability side-effect study, and an explanation of the outputs. In general, the outputs contain two things of interest. The first is statistical information, shown by "stat[KEY] = OCCURRANCES" in the output such as the number of examined repositories. The second - for the exception, sync, and I/O effects - is the method pair effect information denoted by lines starting with "out[] =". The value is a #-delimited list with the following format:
  1. GitHub project name
  2. Method name for the method pair
  3. Subclass name
  4. Supermethod effect
  5. Submethod effect
  6. Category for effect difference
  7. Further information dependent on the analysis
